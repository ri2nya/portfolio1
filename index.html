about me

Maki Ishikawa

石川　真希

はじめまして。石川真希と申します。 現在、未経験からIT業界への転職を目指して、独学でプログラミングを学んでいます。 前職では、海外向け接着剤の開発・改良に携わっており、主に物性評価や実験業務を担当してきました。 実験誤差をできる限り減らす工夫や、効率よく進めるための手順改善などを通じて、着実に品質向上に取り組んできました。 仕事をする中で「自分で何かを作り出す仕事がしたい」という気持ちが芽生え、 もともと好きだったパソコンを活かせるIT分野に強く惹かれるようになりました。 今はHTML/CSSを中心に、Web制作の基礎を学んでいます。 自分の手で形にできることの楽しさを感じながら、一歩ずつ成長中です。

skill

HTML

Progeteで学習。基本的なタグや文法を理解し、自己紹介ページやポートフォリオなどの静的なWebページを作成できます。動くページ作成に向けて学習中です。

CSS

文字の装飾、余白、背景などのスタイル指定ができ、HTMLと組み合わせて簡単なWebページの見た目を整えることができます。メディアクエリは学習中です。

JavaScript

現在Progateを使って学習中です。変数・関数・条件分岐など基本的な文法を理解しながら、Webページへの簡単な動きの実装を目指しています。

まだ学習中の部分も多いですが、毎日できることが増えていくことがとても楽しいです。 新しい動きや、自分が形にしたいものを思い通りに実現できたときは、本当に嬉しくやりがいを感じます。 これからも「やってみる精神」を忘れずに、いろいろな言語や作品作りにチャレンジしていきたいと考えています。
